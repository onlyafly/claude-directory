# Find Advanced Code Smells

Find 3 advanced code smells that could be fixed.

## Steps

1. Identify 3 advanced code smells. An advanced code smell could be:
   - Over-engineered subsystems (YAGNI)
   - Opportunities where your code could have used a third-party library
   - Other broad, system-level concerns

## Notes

When looking for these smells, be creative and think outside the box.
